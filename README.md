# GSoC 2026 — QML-HEP Evaluation Tasks

Evaluation tasks for the "Equivariant Quantum Neural Networks for HEP Analysis at the LHC" project.

- **Task I** — Quantum circuits with Cirq (Hadamard, CNOT, SWAP, swap test)
- **Task II** — Quark/Gluon jet classification with GCN vs GAT
- **Task III** — Commentary on symmetry-aware quantum circuits for HEP
- **Task VII** — Z₂×Z₂ equivariant QNN implementation with PennyLane
